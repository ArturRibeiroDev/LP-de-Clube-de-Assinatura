# 📚 LP Clube de Assinatura

Landing page para clube de assinatura de livros com CSS modular, animações e design responsivo.

## 🚀 Tecnologias

| Tecnologia | Função |
|---|---|
| **HTML5** | Estrutura semântica |
| **CSS3** | Estilização modular por seção |
| **JavaScript** | Interatividade |

## ✨ Funcionalidades

- ✅ CSS modular com arquivos separados por seção (hero, cards, footer, section-book)
- ✅ Design responsivo
- ✅ Elementos decorativos com SVG e imagens
- ✅ Seção de pricing com planos
- ✅ Seção hero com layout visual impactante

## 🏗️ Estrutura

```
/styles
├── global.css      # Reset e variáveis
├── index.css       # Importações
├── hero.css        # Seção hero
├── cards.css       # Cards de destaques
├── footer.css      # Rodapé
├── section-book.css # Seção do livro
```

## 🚦 Como Rodar

Abra `index.html` no navegador. Não requer servidor.

## 💡 O que foi aprendido

- Arquitetura CSS modular (um arquivo por componente/seção)
- Layouts complexos com CSS Grid e Flexbox
- Hierarquia visual com tipografia e espaçamento
